### P0-2 `tests/test_audit_guard.py::test_check_15_pypi_deps_exist` 串行访问 PyPI，超时

**症状：** 单测 10s+ 在本地反复超时。  
**根因：** `audit_guard.check_15_pypi_deps_exist()` 串行访问 PyPI 30 次，每次 `urllib.request.urlopen(url, timeout=5)`。在网络受限环境（CI、本地无代理）下，单次可能 5s+，30 次累计 >10s。  
**CI 影响：** 虽然 CI 设 `timeout=120`，但单测 `timeout=10`（用户本地）会失败。

**修复方案：**
1. 在测试中 mock `urllib.request.urlopen` 返回 200，避免真实网络。
2. 或给该测试单独 `@pytest.mark.network`，CI 默认 skip。

## P1 — 重要改进

### P1-1 大量 "test theater" — `try: ... except Exception: pass` 空断言

`grep "except Exception:\s*pass"` 命中 **30+ 测试文件**，模式形如：

```python
def test_paper_reader_callable(self):
    fn = getattr(mod, "cmd_read", None)
    if fn is None: pytest.skip("not present")
    try:
        assert callable(fn)
    except Exception:
        pass
```

**风险：** 这种测试无论代码是否正确都会 "PASS"，**没有真正覆盖业务逻辑**。表面覆盖率上升，但实际 bug 漏检。

**建议：** 选 5-10 个最有代表性的文件（`test_paper_reader_exec.py`, `test_pipeline_builder_exec.py`, `test_generate_docx_tables_deep.py` 等），把 `except: pass` 替换为 `pytest.xfail(reason="...")` 或真正的断言。

### P1-2 `test_core_literature_vector_store.py` 全部测试是空断言

10 个测试全是 `try: ... except Exception: pass`，**测试通过率为 100% 但覆盖率为 0%**。

**建议：** 删除整个文件（已被 `test_literature_vector_store_unit.py` 全面覆盖），或重写为有效断言。

### P1-3 dead code：5927 个函数定义，缺少外部调用分析

源码中 ~110 个 `PublicClass.method` 仅在自身文件内被调用（典型例子：`AIRouter.bridge`、`AIRouter.cache` 等 properties）。这些可能是：
- 公共 API（保留）
- 配置入口（保留）
- 真的 dead code

**建议：** 用 `vulture scripts/` 或 `pyflakes --select F401,F841` 跑一次，找出真正 dead code。

### P1-4 ruff F841 / S110 仍允许在 tests/ 忽略

`pyproject.toml` 显式忽略 `tests/*.py = ["F841"]`，导致 `var = (call)` 留下裸表达式。虽然不是 bug，但 30+ 测试文件中存在该模式。

### P1-5 `os.getenv` + timeout 组合：5 个网络调用缺少 session reuse

`scripts/research_framework/a_share_variables.py:1158` 等处直接 `requests.get(..., timeout=5)`，未复用 Session，TLS 握手重复耗时。

**建议：** 改用 `requests.Session()` + `HTTPAdapter(pool_connections=10)`。

### P1-6 顶层 `git status` 还会报 `.github/social-preview.png`、`docs/assets/social-preview.png` 修改

`social-preview.png` 已被 gitignore，但本地有差异（61964 vs 62230 bytes），可能是用户手动画过。
